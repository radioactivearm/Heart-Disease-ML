Heart Disease Prediction


This is a webpage made by group 15 members Ainash Toleu, Andy McRae and Jeannaej Yambing for Project 3 at University of Minnesota Data Visualization and Analytics Boot Camp.


Project overview

In this project deep learning was used to create a model that predicts if one has heart disease given an input of 11 features.

For the project we used Heart Disease Dataset found on kaggle.com. The dataset has 11 attributes and 1 target, such as patient age, sex, chest pain type and etc.

The website gets input from user and use a neural network to predict whether they are at risk for heart disease or not. To collect random example data we used this dataset from Kaggle.

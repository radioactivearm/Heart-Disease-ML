Heart Disease Prediction

This is a webpage made by group 15 members Ainash Toleu, Andy McRae and Jeannaej Yambing for Project 3 at University of Minnesota Data Visualization and Analytics Boot Camp.


Project overview

For the project we used Heart Disease Dataset found on kaggle.com. The dataset has 14 attributes, such as patient age, sex, chest pain type and etc.

The website will get input from user and use a neural network to predict whether they are at risk for a heart disease
